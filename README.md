# hci-project-NutrivisonAi
# AI Nutrition Assistant (HCI Project)

## Overview
This project is an AI-based nutrition assistant built as part of our Human Computer Interaction (HCI) coursework. The main goal was to design a system that feels simple and intuitive for users who want to manage their diet in a smarter way.

It helps users decide whether they should focus on weight loss, weight gain, or maintaining their current health based on their BMI, while also making use of the food items they already have.

---

## Features

### 🧮 BMI-Based Guidance
- Users enter basic details like height and weight  
- The system calculates BMI  
- Based on that, it suggests whether the user should lose, gain, or maintain weight  

### 📸 Grocery Image Input
- Users can upload or click a picture of their groceries  
- The system (conceptually using AI) identifies the ingredients available  

### 🍽️ Recipe Suggestions
- Generates recipes based on the available ingredients  
- Provides:
  - Step-by-step instructions  
  - Nutritional information like calories, protein, and carbs  

### 🎥 YouTube Support
- Each recipe also includes a YouTube link  
- Helps users follow along more easily while cooking
